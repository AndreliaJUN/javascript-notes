# 基本类型和引用类型的值
> - 基本类型就是简单的数据段
> - 引用类型的值是保存在内存中的对象
>  - JavaScript不允许直接访问内存中的位置
>  - 也就是说不能直接操作对象的内存空间
>  - 操作对象时，实际上是操作的对象的引用而不是实际的对象
## 动态的属性
- 可以给对象添加属性但是不能给基本类型（undefined/null/boolean/number/String）添加属性
  
## 复制变量值
### 1.复制基本类型
```javascript
var num1=5;
var num2=num1;
```
- 这两个变量可以参与任何操作而不会互相影响
- 两个变量相互独立
